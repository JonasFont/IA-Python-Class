🐍 Python com IA - Curso Prático
Este repositório contém o material e os códigos desenvolvidos durante as 8 aulas do curso de Python com Inteligência Artificial. O objetivo deste projeto é documentar o aprendizado em lógica de programação, bibliotecas de dados e criação de modelos de Machine Learning.

🚀 O que foi aprendido?
Ao longo das aulas, exploramos desde os conceitos básicos de versionamento de código até a criação de redes neurais.
Versionamento: Uso do Git e GitHub Desktop para organizar e salvar o progresso do projeto.
Fundamentos: Lógica de programação e manipulação de dados com Python.IA e Deep Learning: Implementação de modelos sequenciais para tarefas de classificação.

📚 Conteúdo das AulasAula 
1: Introdução ao GitHub - Configuração de ambiente e primeiro commit.
Aula 2: Lógica de Programação - Sequência de instruções e regras de execução.Aula 3: Manipulação de Dados - Introdução ao NumPy e Pandas.
Aula 4: Preparação de Dados para IA - Uso do Scikit-learn para divisão de treino e teste.
Aula 5: Redes Neurais com TensorFlow - Criação de modelos sequenciais e camadas densas.
Aula 6: Funções de Ativação - Aplicação de ReLU e Softmax em problemas de classificação.
Aula 7: Treinamento de Modelos - Compilação com otimizador Adam e monitoramento de perda (loss).
Aula 8: Avaliação e Resultados - Testes práticos, como o script de identificação de cores por RGB.
